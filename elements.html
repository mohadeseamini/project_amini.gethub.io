My Project:

My latest project is the implementation of a todo list application. In this application, as a user, after logging in, we can define tasks for our daily activities, add them to our specific plans, and manage them.

As a user, you first create a weekly or daily plan, etc., and then add tasks to that plan. When tasks are completed, you can mark them as done. A user can have multiple plans, and each plan can contain multiple tasks. Tasks are our daily activities, which have specific priorities and deadlines for completion. They should be sorted based on priority and deadline dates. Additionally, our application has an admin who can manage user accounts.

important points

In this application, each function is implemented using a single function. The main function is not cluttered, and variables have meaningful names. As a result, reading and using the code is easy and possible for all users.

int finduser(vector &users, string &username) {
    for (int i = 0; i < users.size(); i++) {
        if (users[i].username == username) {
            return i;
        }
    }
    return -1;
};
